Portfólio de projetos

Sistema CMMS

Sistema criado com objetivo de centralizar todos os insights e dados de manutenção, aonde é gerados alertas de manutenção, programação de check-lists, controle de custos de manutenção e demais controles utilizando metricas de manutenção como MTTF, MTBF e taxa de disponibilidade.

Tecnologias: Excel, Visual Basic

Manutenção com QR-Code

O projeto consiste basicamente em Qr-Codes instalados em cada máquina aonde o mecânico após realizar o serviço, deve efetuar a leitura e registrar o serviço diretamente no Whatsapp aonde posteriormente essas mensagens eram importadas para o CMMS.

Tecnologias: Whatsapp, Python, Excel

Relógio Eletrônico de Ponto

Projeto desenvolvido para deinir as rotas dos vigias noturnos, foi criado 5 relógios eletronicos aonde cada vigia tem o seu cartão e devem efetuar leituras periódicas, isso define sua rota aonde poderia ser consultada pelo seu gestor posteriormente.

Tecnologias: RFID, ESP32, JavaScript, MySQL

Aplicativo Mobille

Aplicativo desenvolvido para melhorar diversas operações como conferencias, inventários, controles de estoque, WMS e até mesmo entregas de EPI, a criação do AppKadesh foi um grande salto na automação industrial reduzindo custos e aumentando a produtividade.

Tecnologias: Kotlin, Python, Go, MySQL

Sistema de Compras

Sitema completo, desenvolvido para o departamento de compras, integrado diretamente com o sistema ERP da empresa (TOTVS Protheus) com dados essenciais como média de consumo, histórico de compra, preditivo de disponibilidade em dias e projeção de consumo.

Tecnologias: Excel, Visual Basic, Power Query, SQL Server

Sistema PDV

Sistema de PDV para controle de saidas de estoque no almoxárifado, alem desse desenvolvimento, foi realizado um planejamento robusto de padronização, como classificação de produtos por bloco, endereçamento e etiquetamento unitário de cada produto.

Tecnologias: HTML, CSS, Python

Explosão de Matéria Prima

Controle desenvolvido em excel que integra o controle o planejamento e controle de produção, com o sistema de compras e controladoria , consiste em gerar demanda para o departamento de compras e alertas para a controladoria com base na programação da produção.

Tecnologias: Excel, Visual Basic, MySQL

Controle de Produto Acabado

Controle criado para gerenciar as demandas de produtos acabados, com base no histórico de saida e também com controle de estoque atualizado, isso contribui para programação de planos de produção, para sempre ter os produtos com maior rotação à pronta entrega.

Tecnologias: Excel, Visual Basic, Power Query

Controle de Produtividade por Demanda

Desenvolvimento de sistema para controle de demanda de produção Demanda x Capacidade produtiva aonde o Diretor industrial pode acompanhar e fazer simulações de aumento ou dimunuição de capacidade de acordo com a sua demanda atual.

Tecnologias: Excel, Visual Basic, Power Query
